Sub Window_Onload

  MSComm.RThreshold = 1
  MSComm.SThreshold = 1
  MSComm.CommPort = 11
  If Not MSComm1.PortOpen Then : MSComm1.PortOpen = True : End If
  While Not MSComm1.PortOpen : Wend
End Sub

  Select Case MSComm.CommEvent
  Case comEvSend '1 .output
  Case comEvReceive '2 .input
    Document.writeln MSComm1.Input
  End Select
End Sub

Re: HTA: не найден объект Wscript

По первой части отвечу я, по второй - подожду ответа коллег.

Вы не должны хотеть использовать Sleep  в HTA. Sleep - метод, который замораживает исполнение приложения на заданный промежуток времени. То есть все это время приложение будет "висеть" не отвечая на запросы пользователя. Это плохая практика, так как HTA - это интерактивное приложение. Возможно Вы хотели бы объяснить свое желание использовать этот метод необходимостью "отложить на будущее" выполнение некоторой части кода. Для этих целей Вам следует использовать методы setTimeout/setInterval глобального объекта window.

В теме заявлены оба поддерживаемых языка - VBScript, JScript. Я предпочитаю в HTML/HTA использовать JavaScript, поэтому мои цитаты и примеры будут только на нем.

Синтаксис: window.setTimeout(выражение, интервал)
           window.setTimeout(функция, интервал, арг1?, …, аргN?)
Аргументы: выражение — строковое значение
           функция — имя функции
           арг1, …, аргN — значения аргументов функции
           интервал — числовое значение
Результат: целое значение
Этот метод вызывает разовое выполнение определенного действия, когда истечет интервал, заданный в миллисекундах. Действие задается либо строкой, содержащей выражение JavaScript, либо именем функции с необязательным набором аргументов. Метод возвращает целое значение (идентификатор таймера), которое может быть передано методу clearTimeout для остановки данного таймера, т. е. для отмены выполнения заданного действия. Действие будет выполнено по истечении заданного интервала времени, если до этого не будет вызван метод clearTimeout или закрыто данное окно.

<div id="clock"></div><!-- Сюда будут записываться данные -->

// функция, которая определяет текущее время (часы:минуты:секунды)
function ticker()
{
	// Высчитываем время
	var now = new Date();
	var h = now.getHours();
	var m = now.getMinutes();
	var s = now.getSeconds();

	var clock = document.getElementById('clock');
	clock.innerHTML = h + ':' + m + ':' + s;

	setTimeout(ticker, 1000);
};

// вызываем таймер
ticker();

</script>
</body>
</html>

КОД: Копировать Выделить всё

function clickEvent(e)
{
    e = e || event;
    alert(e.srcElement.innerHTML);
};

var text = document.createTextNode('Lorem ipsum');
var para = document.createElement('p');
para.attachEvent('onclick', clickEvent);
para.appendChild(text);

document.body.appendChild(para);

Примерно также делает популярный фреймворк jQuery.

WScript.StdOut.WriteLine "Введите значение переменной a"
Do
    a = WScript.StdIn.ReadLine
    if Not isNumeric(a) Then
        WScript.StdOut.WriteLine "Введено не числовое значение ! Повторите ввод."

        Exit Do
    End if
Loop

        WScript.StdOut.WriteLine "Введено не числовое значение ! Повторите ввод."
    Else
        Exit Do
    End if
